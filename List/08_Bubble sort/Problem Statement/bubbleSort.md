## **Bubble sort**
Given an array A having N positive integers. Sort the array A using bubble sort.

#### **Sample Input**
	input: N = 5
	A : 5 1 6 3 7

#### **Sample Output**
	A: 1 3 5 6 7

#### **Sample Explanation**
Self explanatory.

#### **Expected Time Complexity**
__O__(N*N)

#### **Expected Space Complexity**
__O__(1), i.e., constant space complexity.

#### **Constraints**
	1 <= N <= 1000
